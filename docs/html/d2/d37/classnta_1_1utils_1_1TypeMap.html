Detailed Description

A map from types to values of the given type

Note: Only one value of a given type may be stored Note: This class does not take ownership of any stored pointers It is up to the client to delete them if necessary. Warning: For (most) non-pointer types, this class allocates memory on on the heap when insert is called. Because of type erasure the inserted value's destructor will never be called, so this map cannot store (most) types with non-trivial destructors.
