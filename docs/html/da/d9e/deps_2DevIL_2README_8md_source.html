9 What is it?
10 -----------
11 
12 DevIL is an Open Source image library whose distribution is done under the
13 terms of the GNU LGPL license. See the COPYING file for more details.
14 DevIL offers you a simple way to implement loading, manipulating, filtering,
15 converting, displaying, saving from/to several different image formats in your
16 own project.

42 Why the hell another image library?
43 -----------------------------------
44 
45 I have never seen an image library that can do everything DevIL does. Sure,
46 various different libraries can do part of what DevIL can do as well or even
47 better, but I wanted a simple to use library that encompassed all of these
48 features. I also wanted an extremely portable image library that could be used
49 from a variety of languages and utilized the OpenGL syntax.

59 Errors:
60 -------
61 
62 All errors generated inside DevIL, along with illegal parameters passed to
63 DevIL functions are caught and passed to ilSetError(), an internal library
64 function. The calling program can call ilGetError() to get the value of the
65 error generated. Error types are defined in il.h, using the 0x501 - 0x5FF
66 range. ilGetError() will return 0 (IL_NO_ERROR) if no error has occurred.

69 Basic Usage:
70 ------
71 
72 This demonstrates loading an image through DevIL for OpenGL. Don't forget to
73 call ilInit before you before you do anything:
74 
75 #include <IL/il.h>
76 #include <IL/ilu.h>
77 #include <IL/ilut.h>
78 
79 ...
80 
81 ILuint devilError;
82 
83 
84 ilInit();
85 
86 devilError = ilGetError();
87 
88 if (devilError != IL_NO_ERROR) {
89  printf ("Devil Error (ilInit: %s\n", iluGetErrorString (devilError));
90  exit (2);
91 }
92 
93 ....
94 
95 ILuint devilID;
96 
97 
98 ilGenImages(1, &devilID);
99 ilBindImage(devilID);
100 ilLoadImage("default1.tga"); // Loads into the current bound image
101 devilError = ilGetError();
102 
103 if (devilError != IL_NO_ERROR) {
104  printf ("Devil Error (ilLoadImage: %s\n", iluGetErrorString (devilError));
105  exit (2);
106 }
107 
108 ....
109 
110 ilutRenderer(IL_OPENGL); // Switch the renderer
111 
112 ....
113 
114 GLuint openglID, openglError;
115 
116 
117 openglID = ilutGLBindTexImage(); // This generates the texture for you
118 devilError = ilGetError();
119 
120 if (devilError != IL_NO_ERROR) {
121  printf ("Error: %s\n", iluGetErrorString (devilError));
122  exit (2);
123 }
124 
125 if (openglError != GL_NO_ERROR) {
126  printf ("Opengl Error (ilutGLBindTexImage): %s\n", gluGetErrorString (openglError));
127  exit (2);
128 }
129 
130 
131 
132 // Make sure to close the image when you are done with it (though DevIL
133 // automatically deletes them when the program exits):
134 
135 glDeleteTextures(1, &openglID);
136 ilDeleteImages (1, &devilID);

139 More Examples:
140 ---------
141 
142 The TestIL project is included to test features of DevIL.
143 
144 DevIL includes a project called GLTest. This is a simple test of DevIL's
145 capabilities. All it does it load any image and displays it in a window
146 created by FreeGlut, which is available on http://freeglut.sourceforge.net. It
147 is also included to let the user have an idea of what the library can really
148 be used for.
149 
150 Several other test projects are included to test support with various display
151 APIs. The WindowsTest project is a basic image program that only runs in
152 Windows right now but showcases several of DevIL's features through various
153 menus.
154 
155 If you want more in-depth tutorials, you can find them on
156 http://openil.sf.net, or they may be in your installation under the /examples
157 directory. Documents are also available in the /docs directory.
